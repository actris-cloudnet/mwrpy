File format description
=======================

All MWRpy files use ``NETCDF4_CLASSIC`` data model, i.e., ``HDF5`` file format.


MWR-1C01 file
............

**Dimensions**

+-------------------+
| **Name**          |
+-------------------+
| time              |
+-------------------+
| frequency         |
+-------------------+
| receiver_nb       |
+-------------------+
| ir_wavelength     |
+-------------------+
| bnds              |
+-------------------+
| t_amb_nb          |
+-------------------+
